The Button component can be used to trigger an action.

<Images src={buttonsImg} />

## Usage

```javascript
import { Button } from "thousand-lights";

const ButtonComponent = () => {
  return (
    <Button size="medium" color="primary" appearance="normal">
      Hello World
    </Button>
  );
};
```
