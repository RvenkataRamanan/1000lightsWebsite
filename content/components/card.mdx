The Card component serves as a basic container to display information such as text, images etc.

<Images src={pinterestCardImg} />

## Usage

```javascript
import {
  Card,
  CardHeader,
  CardContent,
  CardFooter,
  Button
} from "thousand-lights";

const CardComponent = () => {
  return (
    <Card>
      <CardHeader
        title="Simple Card"
        description="By Thousand Lights"
        titleStyle={{ color: "red" }}
        descriptionStyle={{ color: "blue" }}
        image={{ uri: "https://i.imgur.com/lceHsT6l.jpg" }}
        rightIcon={
          <Icon
            name="dots-vertical"
            size={25}
            style={{ marginRight: 15 }}
            onPress={() => {}}
          />
        }
        leftIcon={
          <Icon
            name="face"
            size={25}
            style={{ marginRight: 15 }}
            onPress={() => {}}
          />
        }
      />
      <CardContent>
        <Text>
          Lorem Ipsum is simply dummy text of the printing and typesetting industry.
          Lorem Ipsum has been the industry's standard dummy text ever since the
          1500s, when an unknown printer took a galley of type and scrambled it to
          make a type specimen book.
        </Text>
      </CardContent>
      <CardFooter>
        <Button>Get Started</Button>
      </CardFooter>
    </Card>;
  )
}

```

## Props

**The card component can include one or all of CardHeader, CardContent, or CardFooter components.**
