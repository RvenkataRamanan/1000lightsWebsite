BottomSheet based SelectBox for your Ultra-Modern Mobile UI.

## Usage

```javascript
import SelectBox from "thousand-lights";

const SelectBox = () => {

   const data = 
      {
        placeholder: 'Select',
        label: 'Lorem ipsum ',  
      }

    const options =[
      'option1',
      'option2',
      'option3'
    ]

  return (
    <View>
    <SelectBox data={data} options={options} />
    </View>
  );
};

```
