The CardHeader serves as the header for the Card component. It can be used to show a title, subtitle, an image, a left and/or a right icon.

<Images src={cardHeaderImg} />

## Usage

```javascript
import {
  Card,
  CardHeader
} from "thousand-lights";

const CardComponent = () => {
  return (
    <Card>
      <CardHeader
        title="Simple Card"
        description="By Thousand Lights"
        titleStyle={{ color: "red" }}
        descriptionStyle={{ color: "blue" }}
        image={{ uri: "https://i.imgur.com/lceHsT6l.jpg" }}
        rightIcon={
          <Icon
            name="dots-vertical"
            size={25}
            style={{ marginRight: 15 }}
            onPress={() => {}}
          />
        }
        leftIcon={
          <Icon
            name="face"
            size={25}
            style={{ marginRight: 15 }}
            onPress={() => {}}
          />
        }
      />
    </Card>;
  )
}

```
